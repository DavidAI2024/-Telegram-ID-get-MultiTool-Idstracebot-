## 🔍 Overview
**idstracebot** (@idstracebot) is a comprehensive utility bot designed to bridge the gap between standard Telegram usage and technical data. Whether you're investigating a profile, extracting sticker IDs, or generating visual assets, this bot provides a seamless interface for all your Telegram "tracing" needs.

## 🛠 Features Breakdown

### 👤 Profile & OSINT Tools
* **🪪 ID Cards:** Generate a high-quality image containing your profile data (Name, Username, ID).
* **👤 My Info:** Get a full text-based report of your own account details.
* **🆔 User Info:** The core tracing tool. Inspect any user by:
    * Using the dedicated **Search Button** (works even for users *without* a @username).
    * Forwarding a message from them.
    * Sharing a contact.
* **🔍 Link Decoder:** Paste any Telegram link to extract its type, ID, message parameters, and username.

### 🖼️ Media & Style
* **🎨 Sticker Metadata:** Send any sticker to retrieve its **File ID**, **Emoji**, and **Pack Link**.
* **📱 QR Code Generator:** Convert any text or link into a shareable QR code instantly.
* **✏️ Font Styles:** Transform your text into **16 unique Unicode styles** for bios or messages.

### ⚙️ Developer & Technical Utilities
* **🔗 Deep Linker:** Generate all possible Telegram link formats for any given username.
* **🕐 Timestamp Tool:** Convert Unix timestamps to readable dates (and vice-versa) or get the current system time.

## 🚀 Quick Start
1.  Open [@idstracebot](https://t.me/idstracebot) on Telegram.
2.  Tap `/start`.
3.  **To trace someone:** Use the `User Info` button or simply **forward a message** from them to the bot.
4.  **To get sticker data:** Send any sticker directly to the chat.
